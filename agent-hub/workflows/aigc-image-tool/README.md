# 剧情人物设定脚本 生成指南

## 文件结构
每个角色一个独立 JSON + Markdown 文件
```
character-scripts/
├── README.md              # 本文件
├── template.json          # 角色卡模板
├── characters/            # 具体角色
│   ├── lingyun.json       # 凌云（男主）
│   ├── lingyun.md
│   ├── xiaoman.json       # 小满（女主）
│   └── xiaoman.md
└── story-arc/             # 剧情线
    ├── act1.json           # 第一幕
    ├── act2.json           # 第二幕
    └── act3.json           # 第三幕
```

## 角色卡字段说明（参考用户提供的模板）
- 基础信息: 姓名/年龄/身高/体重/生日/星座
- 身体数据: 三围/肩宽/臂长/腿长/足长
- 外貌特征: 发型/发色/瞳色/肤色/脸型
- 服饰风格: 常服/战斗服/礼服 等
- 性格标签: 核心性格关键词
- 背景故事: 简短的背景设定
- 多视角参考: 正面/3/4侧/侧面/背面
- 表情集: 喜/怒/哀/乐/惊/惧/傲/羞 等
- 动作集: 站/坐/走/跑/战斗/施法 等
