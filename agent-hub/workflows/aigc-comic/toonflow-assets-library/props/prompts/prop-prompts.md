## 道具一致性技巧

1. **使用相同种子/参考图**：首次生成满意后固定种子值
2. **关键词保持一致**：每张图包含相同的 {道具名}+{材质}+{主色调}
3. **多角度参考图**：从不同角度生成同一道具，确保设计统一
4. **道具与角色同框**：角色使用道具的图可确保比例统一
