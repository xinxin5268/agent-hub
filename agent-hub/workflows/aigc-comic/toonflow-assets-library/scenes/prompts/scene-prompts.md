# 场景素材库 - 提示词模板集

## 通用场景生成模板

### 全景建立镜头
```
Wide shot of {场景名}, {氛围描述}, {光影}, {时间}, {天气}, epic scale, cinematic, 8k, --ar 16:9
```

### 场景细节镜头
```
Detail shot of {场景名}, focusing on {局部元素}, {光影}, texture details, shallow depth of field, cinematic, 8k
```

### 氛围渲染
```
Atmospheric shot of {场景名}, {氛围}, {色调}, {光影}, moody, cinematic, volumetric lighting, 8k, --ar 16:9
```
