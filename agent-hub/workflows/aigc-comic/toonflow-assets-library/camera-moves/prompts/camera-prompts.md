# 运镜素材库 - 提示词模板集

## 通用运镜模板

### 基础运镜描述结构
```
Camera [运镜方式] toward/around/past [主体], [速度], [情绪] atmosphere, cinematic, [额外效果]
```

### 组合运镜
```
Camera [运镜A] then [运镜B], following [主体] as they [动作], smooth transition, cinematic
```

## 情绪-运镜匹配表

| 情绪 | 推荐运镜 | 速度 | 景别 |
|------|---------|------|------|
| 平静/安宁 | 静止/慢摇 | 极慢 | 全景 |
| 紧张/悬疑 | 慢推 | 缓慢 | 近景→特写 |
| 震惊/冲击 | 快推 | 快速 | 中景→特写 |
| 孤独/渺小 | 拉远 | 缓慢 | 近景→全景 |
| 追逐/行动 | 跟踪/甩镜 | 快速 | 中景 |
| 宏大/史诗 | 升降/环绕 | 缓慢 | 远景 |
| 温柔/亲密 | 环绕 | 缓慢 | 近景 |
| 混乱/战斗 | 手持/甩镜 | 快速 | 中景+晃动 |

## 动作描述模板

### 武术/打斗
```
[角色] performing [武术动作], dynamic pose, [运镜], motion blur, impact lines, action sequence, cinematic
```

### 行走/移动
```
[角色] [walking/running/stride] through [环境], [运镜], [情绪] expression, body in motion, cinematic
```

### 情感动作
```
[角色] [gesture/动作], [情绪] expression, [运镜], intimate moment, soft lighting, cinematic
```

## 分镜描述模板（对齐Toonflow格式）

```
（{画面描述}、{场景}、{关联资产名称}、{时长}、{景别}、{运镜}、{角色动作}、{情绪}、{光影氛围}、{台词}、{音效}、{关联资产ID}）
```
