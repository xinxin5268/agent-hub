# Agent Hub — 多项目会议室方案 v2

## 核心概念
每个项目是一个独立的"会议室"，项目内的 Agent 在会议室里实时协作。

```
┌─────────────────────────────────────────────────────┐
│                    Agent Hub                         │
├──────────┬──────────┬──────────┬────────────────────┤
│ 项目 A    │ 项目 B    │ 项目 C    │ 全局视图           │
│ 会议室     │ 会议室     │ 会议室     │ (所有 Agent 概览)  │
│ ┌──────┐ │ ┌──────┐ │ ┌──────┐ │                    │
│ │小宝   │ │ │小宝   │ │ │小聪   │ │                    │
│ │小聪   │ │ │Hermes │ │ │OpenCode│ │                    │
│ │OpenCode│ │ │OpenCode│ │ │Hermes │ │                    │
│ └──────┘ │ └──────┘ │ └──────┘ │                    │
└──────────┴──────────┴──────────┴────────────────────┘
```

## 需要实现的功能

### 1. 多项目管理
- 左侧项目列表（类似 Slack 的频道列表）
- 每个项目可创建/删除/重命名
- 每个项目独立管理 Agent 成员
- 切换项目 = 切换会议室

### 2. 会议室视图
- **Agent 视频卡片网格** — 每个 Agent 一张卡片
  - 头像/图标（按类型区分）
  - 名称 + 角色标签
  - 状态灯（在线/忙碌/离线/发言中）
  - 音频波浪动画（发言时跳动）
  - 当前任务/状态文字
- **协作链路 DAG** — 显示 Agent 间的执行流程
  - 节点 = Agent（颜色按状态）
  - 箭头 = 消息/任务流向
  - 实时更新

### 3. Agent 实时通信
- 会议室内的消息广播（所有成员可见）
- 私聊（@某个 Agent）
- 指令执行（向 Agent 下发任务）
- 消息有发送者头像、时间戳、状态（已发送/已读/已完成）

### 4. 实时动画
- Agent 状态变化：online→busy 平滑过渡
- 发言时：卡片边框发光 + 音频波浪
- 新消息：滑入动画
- DAG 链路：节点颜色渐变
- 协作完成：成功动画（绿光扫过）

### 5. 全局视图
- 所有项目的 Agent 概览
- 跨项目 Agent 调度
- 全局活动流

## 技术实现

### 注册中心扩展
```
消息路由:
  Agent A → 注册中心 → Agent B (WebSocket)
                    → 所有 UI 客户端广播

事件类型:
  room:join      — Agent 加入会议室
  room:leave     — Agent 离开
  room:message   — 消息
  room:status    — 状态变更
  room:task      — 任务分配
  room:chain     — 协作链路更新
```

### UI 组件结构
```
src/features/studio/
├── StudioPanel.tsx     # 主面板（项目列表 + 会议室）
├── ProjectList.tsx     # 项目侧边栏
├── MeetingRoom.tsx     # 会议室主视图
├── AgentCard.tsx       # Agent 视频卡片（含动画）
├── CollaborationDAG.tsx # 协作链路 DAG
├── MessageStream.tsx   # 实时消息流
├── CommandInput.tsx    # 指令输入
└── GlobalOverview.tsx  # 全局概览
```

### 动画实现
- CSS transitions + Tailwind 动画类
- 音频波浪：CSS @keyframes（已有 .audio-bars）
- DAG：SVG + 状态驱动的颜色变化
- 卡片发光：box-shadow 动画
