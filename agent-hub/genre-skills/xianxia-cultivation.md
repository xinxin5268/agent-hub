# 玄幻修仙题材 Skill

> 定义玄幻修仙题材的写作规则、模板和平台适配

## 1. 题材定义

**核心特征**: 以修炼变强为主线，融合东方神话、道家思想、武侠元素
**读者预期**: 升级爽感、奇遇探索、门派争斗、天劫飞升
**典型元素**: 灵气、丹田、元婴、法宝、丹药、阵法、天劫、秘境

**分级定位**:
| 层级 | 代表作品 | 特点 |
|------|----------|------|
| 小白文 | 《凡人修仙传》《仙逆》 | 升级快、爽点密集 |
| 经典派 | 《诛仙》《搜神记》 | 文笔细腻、情感丰富 |
| 创新派 | 《诡秘之主》（克系修仙） | 跨界融合 |
| 严肃派 | 《道诡异仙》 | 黑暗修仙、哲学深度 |

## 2. 世界观模板

### 修炼体系（标准九境）

| 境界 | 寿命 | 能力 | 标志 |
|------|------|------|------|
| 练气 | 120年 | 引气入体 | 气感 |
| 筑基 | 200年 | 灵力凝实 | 道基 |
| 金丹 | 500年 | 丹成九转 | 金丹异象 |
| 元婴 | 1000年 | 元婴出窍 | 元神 |
| 化神 | 3000年 | 法则感悟 | 领域 |
| 合体 | 5000年 | 天人合一 | 道韵 |
| 大乘 | 10000年 | 渡劫准备 | 天劫 |
| 渡劫 | 飞升 | 天劫洗礼 | 九九天劫 |
| 飞升 | 永恒 | 仙人 | 仙灵之气 |

### 势力分布

| 势力类型 | 定位 | 冲突点 |
|----------|------|--------|
| 宗门 | 主角成长环境 | 内部争斗、资源分配 |
| 家族 | 血脉传承 | 嫡庶之争、家族兴衰 |
| 散修 | 自由但危险 | 资源匮乏、被欺压 |
| 魔道 | 反派势力 | 正邪对立 |
| 上古遗迹 | 机缘所在 | 争夺宝物 |

### 核心规则
- **天道规则**: 境界越高、突破越难、天劫越强
- **因果报应**: 杀人夺宝有业力，影响心魔劫
- **气运之说**: 主角有大气运，但气运可消耗
- **资源竞争**: 灵气有限，资源决定修炼速度

## 3. 角色模板

### 主角模板

| 类型 | 人设 | 弧光 |
|------|------|------|
| 废材逆袭 | 灵根差/丹田废 | 从最底层一步步爬起 |
| 重生者 | 带着前世记忆重修 | 弥补遗憾、快意恩仇 |
| 穿越者 | 现代灵魂穿越到修仙界 | 用现代思维降维打击 |
| 天才 | 天生道体/圣体 | 怀璧其罪、举世皆敌 |

### 配角模板

| 类型 | 功能 | 注意事项 |
|------|------|----------|
| 师父 | 传道授业 | 不要抢主角风头，适时退场 |
| 红颜 | 感情线 | 1-3个，不要后宫太多 |
| 兄弟 | 辅助/牺牲 | 推动主角成长的关键角色 |
| 宿敌 | 持续对抗 | 要有层次，不是脸谱化反派 |
| 老爷爷 | 金手指 | 合理限制，不能全知全能 |

## 4. 情节结构

### 标准升级流

```
第1-10章：入门（进入宗门/获得传承）
第11-30章：筑基期（初次冲突、获得机缘）
第31-60章：金丹期（宗门大比、秘境探险）
第61-100章：元婴期（外出历练、正邪大战）
第101-150章：化神期（上古遗迹、域外天魔）
第151-200章：合体期（势力大战、飞升准备）
第201-250章：渡劫飞升
```

### 关键节拍

| 节拍 | 位置 | 内容 |
|------|------|------|
| 金手指获得 | 1-5章 | 让主角有特殊优势 |
| 首次碾压 | 10-15章 | 让主角展示实力 |
| 第一次大败 | 30-40章 | 让主角认识到不足 |
| 秘境探险 | 50-70章 | 获得关键机缘 |
| 宗门危机 | 80-100章 | 主角挺身而出 |
| 飞升雷劫 | 尾声 | 最后的高潮 |

## 5. 写法规则

### 语言风格
- 多用古风词汇：道、法、天、地、玄、黄、宇、宙
- 境界名称统一，不混用不同体系
- 打斗描写注重"势"的对比（气势压制→招式对抗→法宝碰撞）
- 修炼过程要有"突破感"（冲关→瓶颈→顿悟→突破）

### 常见陷阱
- ❌ 主角升级太快（失去成长感）
- ❌ 反派太弱（没有压迫感）
- ❌ 奇遇太多（读者麻木）
- ❌ 后宫管理失控（人物扁平）
- ✅ 每50章让主角经历一次真正的失败

### 爽点设计
| 爽点 | 间隔 | 示例 |
|------|------|------|
| 升级突破 | 每10-15章 | 突破瓶颈、境界提升 |
| 打脸反派 | 每20-30章 | 碾压看不起主角的人 |
| 获得宝物 | 每30-50章 | 秘境、传承、炼器 |
| 实力展示 | 每40-60章 | 宗门大比、灭杀强敌 |

## 6. 平台适配

| 平台 | 要求 | 禁忌 |
|------|------|------|
| 起点中文网 | 男频向、升级爽、字数长 | 涉政、涉黄、宣扬邪教 |
| 纵横中文网 | 类似起点 | 同起点 |
| 飞卢小说 | 脑洞大、节奏快 | 抄袭、低俗 |
| 番茄小说 | 免费阅读、黄金3章 | 灌水、质量差 |
| 晋江文学城 | 感情线丰富 | 男频向难火 |
| 海外 | 翻译版 | 文化概念需加注释 |

## 7. 示例参考

### 经典开篇
```
《凡人修仙传》: "韩立深吸一口气，感受着体内微弱的气感..."
特点：从最底层开始，让读者有代入感

《仙逆》: "王林呆呆地看着手中的天逆珠..."
特点：开局给金手指，但不解释来源
```

### 经典打斗
```
"剑气纵横三万里，一剑光寒十九洲。"
特点：用意境代替具体动作
```
