# 现代都市言情题材 Skill

> 定义现代都市言情题材的写作规则、模板和平台适配

## 1. 题材定义

**核心特征**: 以都市为背景，聚焦人物情感关系和成长
**读者预期**: 情感共鸣、人设魅力、甜蜜或虐心的情感冲突
**典型元素**: 职场、豪门、破镜重圆、先婚后爱、暗恋成真

**分级定位**:
| 层级 | 代表作品 | 特点 |
|------|----------|------|
| 甜宠文 | 《微微一笑很倾城》 | 全程撒糖、无虐点 |
| 虐恋文 | 《何以笙箫默》 | 有分离、有误会、有成长 |
| 现实向 | 《欢乐颂》 | 贴近真实生活 |
| 悬疑+言情 | 《他来了请闭眼》 | 双线并行 |

## 2. 世界观模板

### 都市背景设定

| 背景类型 | 场景素材 | 氛围 |
|----------|----------|------|
| 一线城市 | 陆家嘴、国贸、CBD、三里屯 | 快节奏、压力大 |
| 二线城市 | 西湖、锦里、老城区 | 慢生活、人情味 |
| 海外都市 | 纽约、东京、巴黎 | 异国恋、文化冲突 |
| 校园 | 大学校园、图书馆、操场 | 青春、纯粹 |

### 核心设定维度
- **时间**: 当代（2020-2030）或特定年代（90年代怀旧）
- **阶级**: 豪门/中产/草根，阶级差异是主要冲突源
- **职业**: 霸总/医生/律师/设计师/教师/作家/程序员（影响人物气质）
- **城市气质**: 上海的精致、北京的豪爽、深圳的拼搏、成都的悠闲

### 情感阶段
```
相遇 → 好感 → 暧昧 → 表白 → 热恋 → 冲突 → 分离 → 成长 → 重逢 → 圆满
（每个阶段约占总篇幅的10-15%）
```

## 3. 角色模板

### 男主模板

| 类型 | 人设 | 苏点 |
|------|------|------|
| 霸总 | 高冷、富有、掌控欲强 | 只对女主温柔 |
| 暖男 | 温柔、体贴、忠诚 | 默默付出不求回报 |
| 学霸 | 智商高、情商低 | 为爱成长、笨拙的浪漫 |
| 痞帅 | 玩世不恭、有故事 | 表面不羁、内心深情 |
| 阳光 | 开朗、运动、正直 | 永远乐观、治愈女主 |

### 女主模板

| 类型 | 人设 | 成长弧 |
|------|------|--------|
| 职场精英 | 独立、能力强 | 学会示弱、接受帮助 |
| 傻白甜 | 单纯、善良 | 成长、不再依赖他人 |
| 御姐 | 成熟、有故事 | 打开心扉、重新相信爱情 |
| 灰姑娘 | 平凡、坚强 | 发现自己的独特价值 |
| 事业型 | 专注事业、感情小白 | 平衡事业与爱情 |

### 关键配角

| 角色 | 功能 | 注意事项 |
|------|------|----------|
| 闺蜜/兄弟 | 助攻、吐槽、情感顾问 | 不要脸谱化工具人 |
| 情敌 | 制造危机、推动主角行动 | 要有魅力、不低俗 |
| 长辈 | 制造家庭矛盾/提供智慧 | 合理动机，不强行反派 |
| 前度 | 制造误会、测试感情 | 不要纠缠太久 |

## 4. 情节结构

### 标准甜宠文结构

```
第1-5章：相遇（偶然相遇/被迫同居/契约婚姻）
第6-10章：好感（日常相处、互生好感）
第11-15章：暧昧（肢体接触、眼神交流、暧昧对话）
第16-20章：冲突（误会/情敌/家庭反对）
第21-25章：分离（虐心桥段、各自成长）
第26-30章：重逢（破镜重圆、HE结局）
```

### 经典桥段库

| 桥段 | 用法 | 频率 |
|------|------|------|
| 意外接吻 | 快速拉近距离 | 开局用1次 |
| 醉酒表白 | 打破暧昧僵局 | 中期用 |
| 英雄救美 | 展示男主能力 | 可多次用 |
| 雨中追逐 | 情感爆发点 | 高潮用 |
| 生病照顾 | 展示温柔一面 | 中前期 |
| 吃醋情节 | 确认感情 | 暧昧期用 |
| 分别告白 | 分离前的情感宣泄 | 分离桥段用 |

### 情感节奏控制

```
前30%：相识、好感建立（轻快节奏）
30-60%：暧昧、确认关系（甜蜜期）
60-80%：冲突、分离（虐心期）
80-100%：成长、重逢、圆满（治愈期）
```

## 5. 写法规则

### 对话风格

| 人设 | 对话特点 | 示例 |
|------|----------|------|
| 霸总 | 简短、命令式、偶尔毒舌 | "上车。""别让我说第二遍。" |
| 暖男 | 温柔、体贴、善解人意 | "冷不冷？我的外套给你。" |
| 傻白甜 | 活泼、直白、有点话痨 | "哇！这个好好吃！你尝尝！" |
| 御姐 | 从容、带刺、有内涵 | "男人的承诺，我听过太多。" |

### 描写重点

| 场景 | 描写重点 | 避免 |
|------|----------|------|
| 初遇 | 外貌第一印象、环境氛围 | 过度外貌描写 |
| 暧昧 | 微表情、肢体语言、心理活动 | 直白表白 |
| 虐心 | 内心独白、回忆闪回 | 过度矫情 |
| 甜蜜 | 细节动作、对话 | 腻到发齁 |

### 反AI味要点
- ❌ 不要每个角色说话都"微微一笑""低声说道"
- ❌ 不要每段描写都有"仿佛""似乎""好像"
- ❌ 不要滥用"宠溺""霸道""温柔"等标签
- ✅ 用具体动作代替情感标签（"他握住她的手"代替"他充满爱意"）

## 6. 平台适配

| 平台 | 要求 | 禁忌 |
|------|------|------|
| 晋江文学城 | 女频向、感情细腻 | 涉黄、未成年恋爱 |
| 起点女生网 | 长文、有主线 | 太过幼稚 |
| 红袖添香 | 古风言情优先 | 抄袭、注水 |
| 番茄小说 | 免费向、节奏快 | 前三章必须抓人 |
| 微信读书 | 出版向、现实题材 | 质量要求高 |
| 海外 | 翻译向 | 文化差异注释 |

## 7. 示例参考

### 经典开篇
```
《何以笙箫默》: "如果世界上曾经有那个人出现过，其他人都会变成将就。"
特点：一句话定调，情感浓度高

《微微一笑很倾城》: "一笑倾城，二笑倾国，三笑倾我心。"
特点：轻松甜蜜的开场
```

### 经典场景
```
雨中告白桥段：
"我爱你。"他说得很轻，像是怕惊动什么。
雨声很大，但她听得很清楚。
```

## 8. 跨文化适配

| 中文元素 | 英文版适配 |
|----------|-----------|
| 相亲 | Blind date/matchmaking |
| 豪门恩怨 | Family feud/Corporate rivalry |
| 先婚后爱 | Marriage of convenience |
| 青梅竹马 | Childhood sweethearts |
| 职场潜规则 | Office politics |

**注意**: 海外言情更注重女性独立和自我成长，中式"霸总宠妻"的套路在海外可能水土不服，需要加入更多女性自主意识。
