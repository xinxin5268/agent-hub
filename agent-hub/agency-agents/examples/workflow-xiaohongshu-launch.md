# 多智能体工作流：小红书品牌推广

> 从 0 到 1 用多个智能体配合完成一次小红书品牌种草推广。

## 场景

你是一个新消费品牌，准备在小红书上做第一波种草推广。预算有限，需要 2 周内完成从策略到执行到复盘的全流程。

## 智能体团队

| 智能体 | 在这个流程中的角色 |
|--------|-------------------|
| 小红书运营 | 制定内容策略和达人合作方案 |
| 品牌守护者 | 确保品牌调性一致 |
| 内容创作者 | 产出种草笔记内容 |
| 数据分析师 | 追踪投放数据、出复盘报告 |
| 增长黑客 | 设计裂变和转化路径 |

## 工作流

### 第一阶段：策略制定（第 1-2 天）

**第 1 步 — 激活小红书运营**

```
激活小红书运营。

品牌：[你的品牌名] — [一句话品牌定位]。
品类：[具体品类，比如护肤/零食/家居]。
目标人群：[年龄、城市、兴趣标签]。
预算：[总预算]，其中达人合作 [X]，信息流投放 [X]。
目标：2 周内笔记曝光 50 万+，进店 UV 5000+。

请给出：
1. 竞品在小红书上的打法分析（找 3-5 个竞品）
2. 内容方向建议（3 个主题方向）
3. 达人合作方案（KOC/KOL 配比、预算分配）
4. 两周内容发布排期
```

**第 2 步 — 激活品牌守护者（同步进行）**

```
激活品牌守护者。

品牌：[你的品牌名]。
品牌调性：[关键词，比如"自然、真实、年轻"]。
视觉规范：[主色调、字体风格、拍摄风格]。
绝对不能出现的：[负面关键词，比如"最便宜"、"绝对有效"]。

请输出一份小红书内容审核清单，给到后续写内容和审达人稿件的时候用。
```

### 第二阶段：内容生产（第 3-7 天）

**第 3 步 — 激活内容创作者**

```
激活内容创作者。

参考小红书运营给出的 3 个内容方向，每个方向写 2 篇种草笔记。

要求：
- 标题用小红书爆款标题公式（数字+痛点+解决方案）
- 正文控制在 300-500 字，口语化，像朋友推荐
- 每篇标注建议配图方向（几张图、拍什么）
- 加上 10-15 个话题标签

注意品牌守护者给的审核清单，不要踩红线。
```

**第 4 步 — 激活增长黑客**

```
激活增长黑客。

在小红书种草内容的基础上，设计转化路径：
1. 笔记 → 评论区引导 → 私信/店铺的转化链路
2. 互动抽奖/打卡活动方案（提升笔记互动量）
3. 老用户晒单返利机制

预算约 [X]，重点是用低成本撬动自然流量。
```

### 第三阶段：投放和优化（第 8-12 天）

**第 5 步 — 再次激活小红书运营**

```
内容已经发布 3 天了，以下是数据：

笔记 A：曝光 8000，点赞 120，收藏 85，评论 30
笔记 B：曝光 15000，点赞 350，收藏 200，评论 65
笔记 C：曝光 3000，点赞 20，收藏 15，评论 5
...

请分析：
1. 哪些内容跑得好，为什么
2. 跑得差的问题在哪
3. 剩余预算的信息流投放建议（加推哪几篇）
4. 是否需要调整后续达人合作方向
```

### 第四阶段：复盘（第 13-14 天）

**第 6 步 — 激活数据分析师**

```
激活数据分析师。

这是两周小红书推广的完整数据：

总曝光：[X]
总互动（点赞+收藏+评论）：[X]
进店 UV：[X]
成交订单：[X]
总花费：[X]

各笔记明细数据：[贴数据表]
各达人数据：[贴数据表]

请输出复盘报告：
1. 核心指标达成情况 vs 目标
2. ROI 分析
3. 内容类型效果对比
4. 达人合作效果对比
5. 下一轮推广的优化建议
```

## 关键交接点

| 从 | 到 | 交接内容 |
|----|-----|---------|
| 小红书运营 | 内容创作者 | 内容方向、达人 brief |
| 品牌守护者 | 内容创作者 | 内容审核清单 |
| 内容创作者 | 小红书运营 | 成稿，等审核和发布 |
| 小红书运营 | 数据分析师 | 投放数据 |
| 数据分析师 | 小红书运营 | 复盘报告、优化建议 |

## 预期产出

- 6-10 篇种草笔记（含达人合作内容）
- 1 套品牌小红书内容规范
- 1 份完整的推广复盘报告
- 下一轮推广的可复用 SOP
